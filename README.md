## ODIN-RECIPES

Basic recipe pages with links and images.

### What I Learned

All folders, files, and images in this project were created and organized entirely through the terminal.

Throughout the development process, I refreshed my knowledge of fundamental Git commands.

Additionally, I gained hands-on experience in the following web development concepts:

HTML Structure: Implementing and managing lists within a website.

Media: Adding and positioning images.

Navigation: Creating links for seamless page transitions.

Formatting: Using HTML elements such as `<strong>` and `<em>.`

Workflow & Productivity
I practiced efficient workflow techniques in Visual Studio Code using various keyboard shortcuts, including:

Navigating between files.

Moving and duplicating lines.

Multi-line selection and simultaneous editing.
